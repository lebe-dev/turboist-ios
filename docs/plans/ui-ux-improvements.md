# Доработки UI/UX для iOS-приложения

## Контекст

Turboist iOS — SwiftUI-приложение. Задачи отображаются в нескольких видах (Today, Tomorrow, All, Weekly, Inbox, Backlog). Страницы Сегодня/Завтра используют `DayPartSectionView` с группировкой по фазам дня. Контекстное меню (`TaskContextMenuView`) показывается при долгом нажатии на задачу. Навигационный островок (`ViewSwitcherView`) — плавающая панель с основными разделами.
